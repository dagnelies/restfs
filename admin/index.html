Embed it

You can embedd it into your own website by simply providing an iframe.

<iframe src="https://..."><iframe>

If you want to avoid the login screen, you can use the "indirect authentication" feature.

First, make a server side POST call to https://.../token?ftp=...&username=...&password=... to obtain an access token.

Then, provide the access token instead of the creditentials in the iframe.

<iframe src="https://.../connect?token=..."><iframe>
